# Link-in-Bio — ARJO

A high-performance, aesthetically pleasing "Link-in-Bio" landing page designed for digital creators. This project features a modern dark-mode aesthetic with glassmorphism effects, interactive elements, and a seamless mobile-first experience.

---

## 🚀 Live Features

* **Glassmorphism Design**: Semi-transparent "glass" cards with backdrop-blur filters for a premium look.
* **Integrated Music Player**: Custom play/pause functionality with a CSS-animated pulse ring.
* **Responsive Layout**: Fully optimized for mobile, tablet, and desktop using Tailwind CSS.
* **Scroll Animations**: Smooth entry effects powered by AOS (Animate On Scroll).
* **Functional Contact Form**: A "Send a Letter" section integrated with Formspree for direct-to-email messaging.
* **Lucide Icons**: Clean, consistent SVG iconography.

## 📂 File Structure

To make this work locally, ensure your directory looks like this:

* `index.html` (The main code)
* `assets/`
    * `pfp.png` — Your profile picture
    * `bg.jpg` — Background image
    * `song.mp3` — The background track
    * `instagram.svg`, `youtube-icon.svg`, `discord.svg` — Social icons
    * `in_love.webp` — Profile decoration/avatar frame

---

## ⚙️ Setup & Customization

1.  **Update Social Links**: Change the `href` attributes in the link section to point to your profiles.
2.  **Configure Contact Form**: Replace the Formspree ID `mdaylyoe` with your own unique ID from [Formspree.io](https://formspree.io/).
3.  **Change Music**: Replace `assets/song.mp3` with your own file. The player is currently set to a volume of **30%** (`audio.volume = 0.3`).
4.  **Edit Profile**: Update the `h1` and `p` tags to reflect your name and bio.
